Type

Date

Role

Group Leader and Contributor

This project is an interactive D3.js dashboard exploring the relationship between life expectancy, population change, public expenditure, and demographic structure across countries. It was built as part of an F21DV group project and combines multiple linked charts so the viewer can move from global patterns to country-level detail.

01 / What the project does

The dashboard uses five coordinated views: a choropleth map, scatter plot, donut chart, butterfly chart, and line chart. Filters at the top of the interface update the different visualisations, helping users compare countries and observe how broader demographic and health indicators change together.

02 / Why it belongs in the portfolio

This is a strong portfolio piece because it shows more than analysis alone. It demonstrates visual storytelling, interaction design, data wrangling, and front-end implementation in D3.js, while also highlighting collaboration on a substantial school project.

03 / Recommended presentation strategy

The cleanest setup is to keep the coursework dashboard in its original folder as the live artifact, and feature this page as the case study layer. That gives visitors context first, then lets them jump into the full interactive project when they are ready.
